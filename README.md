Spring Framework를 활용한 MVC 패턴의 모임 커뮤니티 사이트입니다.<br>
'소수'에서는 맛집투어, 스포츠 즐기기, 게임 등의 취미 활동을 소소하게 함께 즐길 인원을 모집할 수 있습니다.<br>
참여한 모임에 대한 리뷰를 남길 수 있으며 별도의 자유게시판도 이용 가능합니다.<br><br><br>
